Como usar Pixton

Para la actividad, se utilizará Pixton a través del enlace proporcionado: https://edu-es.pixton.com/solo/

Utilicen el botón "Registro para Estudiantes" y seleccionen "Sí" cuando se les pregunte si se les ha proporcionado un enlace para unirse a una clase . Luego, ingresen el código 79wpkk creado para la clase y seleccionen "continuar".

 

Para su registro, utilicen la clave de usuario y la contraseña de Google. Se recomienda tener una cuenta de Google especial para la utilización de aplicaciones, independiente de la que usan para asuntos personales.

Al registrarse, tendrán acceso a la aplicación para crear cómics de forma gratuita a través de la asignatura. 

Seleccionen el botón "CREA MI PRIMER CÓMIC" y especifiquen el tema de su historieta.

Al entrar, verán la ventana de edición, que contiene un menú de opciones en el lado izquierdo para desarrollar el cómic: Fondo, Personajes, Objetos, Foco, Palabras, Caras y Acciones. 

Dependiendo del elemento seleccionado en el menú, aparecerán diferentes opciones para utilizar. Haz clic en las siguientes imágenes para acceder a una breve explicación de las opciones básicas en la aplicación.



Una vez diseñado el cómic, podrán guardarlo con el botón "LISTO", ubicado en la parte superior derecha. En el área del cómic, podrán "Editar" y "Publicar". Si seleccionan "Publicar", tendrán acceso a cinco opciones:

Para la actividad, seleccionen "DESCARGAR" y luego "Composición personalizada" o "2 columnas" para generar un archivo en formato de imagen que se bajará al equipo de cómputo.

Además, consideren la opción "Compartir" para generar un enlace al cómic en la red.

En la parte superior izquierda tendrán un icono que los llevará al área personal, donde podrán crear más cómics o regresar al mismo para editarlo.

No olviden cerrar sesión cada vez que terminen de trabajar con la APP, utilizando el icono de engranaje en la parte superior derecha, junto a su nombre.
